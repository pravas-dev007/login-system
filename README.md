# 🔐 Login & Registration System

> A web-based authentication system built using Servlets and JDBC with session management.

---

## 🚀 Project Overview
This project is a simple Login and Registration system where users can create an account and log in.  
It uses Servlets for request handling, JDBC for database interaction, and HTTP Sessions for authentication.

---

## 🛠️ Tech Stack
- Java
- Servlets (J2EE)
- JDBC
- PostgreSQL
- HTML
- Maven

---

## ✨ Features
- User Registration
- User Login
- Session Handling
- Logout Functionality
- Basic Validation

---

## 📂 Project Structure
controller  
dao  
model  
util  

---

## 🔗 Flow
1. Register user  
2. Login user  
3. Create session  
4. Show dashboard  
5. Logout  

---

## ▶️ How to Run
1. Clone project  
2. Open in IDE  
3. Configure DB in DBConnection  
4. Run on Tomcat  
5. Open:
   - /register.html  
   - /login.html
